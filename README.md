# Chrome Extensions ことえり文字変換確定の送信無効化 for ClickUp

MacOSのChromeで、ClickUpのタスク編集時にテキストボックス内で日本語を変換し、Enterキーで変換を確定するとフォーカスが外れてしまい、続けて入力することができなくなる問題に悩んでいたので、暇つぶしに作った変換確定時のEnterキー入力をキャンセルするだけの拡張機能。
